# Employees Salaries Exploratory Data Analysis (EDA)

## Project Overview

This project performs Exploratory Data Analysis (EDA) on employee salary data to uncover insights about salary distributions, job titles, and compensation trends using Python.

## Technologies Used

* Python
* Pandas
* NumPy
* Jupyter Notebook

## Data Cleaning

* Checked missing values and data types.
* Replaced invalid values such as "Not Provided" with NaN.
* Removed rows with missing values.
* Converted columns to appropriate data types.

## Analysis Performed

* Displayed top and bottom records of the dataset.
* Determined the number of rows and columns.
* Checked dataset information and null values.
* Identified unique job titles.
* Found employees with specific job titles.
* Calculated minimum, maximum, and average Base Pay.
* Identified employees with the highest salaries.
* Analyzed average Base Pay by year and job title.
* Determined the most common job titles.

## Key Findings

* Salary distributions varied significantly across job titles.
* Certain job roles had consistently higher average salaries.
* Missing and invalid values required preprocessing before analysis.
* Job title analysis provided insights into compensation patterns.

## Conclusion

This project demonstrates practical skills in data cleaning, preprocessing, filtering, aggregation, and exploratory data analysis using Python and Pandas.
