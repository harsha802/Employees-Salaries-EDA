# Employees Salaries Exploratory Data Analysis (EDA)

## Project Overview

This project performs Exploratory Data Analysis (EDA) on employee salary data to identify compensation patterns, salary distributions, overtime trends, and differences in pay across various job roles using Python.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook

## Data Cleaning

* Checked missing values and data types.
* Replaced invalid values such as `"Not Provided"` with `NaN`.
* Removed rows with missing values.
* Converted columns to appropriate data types.
* Removed duplicate records where necessary.

## Analysis Performed

* Displayed the top and bottom records of the dataset.
* Determined the number of rows and columns.
* Examined dataset information and null values.
* Identified unique job titles.
* Found employees with specific job titles.
* Calculated minimum, maximum, and average Base Pay.
* Identified employees with the highest salaries.
* Analyzed average Base Pay by year and job title.
* Determined the most common job titles.
* Investigated overtime compensation patterns.

## Visualizations

* Salary Distribution Histogram
* Top 5 Job Titles with Highest Average Base Pay
* Top 10 Highest Paying Job Titles
* Overtime Pay Analysis
* Average Total Pay Benefits by Year

## Key Findings

* Leadership positions had the highest average base pay.
* Operational roles earned higher overtime compensation.
* Most employees received moderate salaries, while a few earned exceptionally high compensation.
* Employee compensation varied across different years.
* Job title analysis provided useful insights into compensation patterns.

## Conclusion

This project demonstrates practical skills in data cleaning, preprocessing, filtering, aggregation, data visualization, and exploratory data analysis using Python and Pandas to extract meaningful insights from real-world employee salary data.
